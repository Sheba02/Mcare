# MamaCare+

MamaCare+ is an Expo-based React Native frontend for a maternal and child health companion system.

## What is included

- A fresh Expo app setup ready for Expo Go
- A polished first-pass mobile UI shaped around the product concept
- Four core frontend sections: `Home`, `Tracker`, `Care`, and `Learn`
- Content blocks for reminders, pregnancy tracking, baby care, symptom guidance, and follow-up workflows

## Run locally

```bash
npm install
npx expo start
```

Then scan the QR code with Expo Go on your phone.

## Notes

This first pass focuses on product-facing frontend structure and visual direction. It does not include backend services, authentication, persistence, or offline sync yet.
