# MamaCare+

MamaCare+ is an Expo-based React Native frontend for a maternal and child health companion system.

## What is included

- TypeScript app structure with `screens`, `navigation`, `components`, `theme`, `types`, and `data`
- Root auth flow with splash, welcome, login, register, forgot password, and onboarding screens
- Main product area with dashboard, pregnancy tracker, baby tracker, appointments, learning, profile, and symptom checker
- React Navigation setup for a professional multi-screen mobile experience

## Run locally

```bash
npm install
npx expo start -c
```

Then scan the QR code with Expo Go on your phone.

## Notes

This version is frontend-first. The screen flows are in place, but backend authentication, persistence, APIs, and offline sync still need to be connected in later phases.
